## [0.1.0] - 2025-01-10

### Added
- Initial release of NeoCode
- Core Neovim configuration with Lua
- tmux integration for session management
- Custom statusline and UI components
- LSP support with automatic server installation
- Plugin system based on lazy.nvim
- File explorer with nvim-tree
- Fuzzy finding with Telescope
- Git integration with gitsigns
- Catppuccin theme support
- Installation and session management scripts
- Comprehensive documentation

### Core Features
- **Editor**: Vim-style editing with modern enhancements
- **Sessions**: Persistent tmux sessions with organized layouts
- **LSP**: Language Server Protocol support for multiple languages
- **Plugins**: Modular plugin architecture
- **Themes**: Customizable color schemes
- **Configuration**: YAML/Lua-based settings with hot-reloading
- **Terminal**: Integrated terminal support
- **Git**: Built-in Git operations and status display

### Supported Platforms
- Linux (Ubuntu, Debian, Arch, CentOS)
- macOS
- Windows (via WSL)

### Installation Methods
- Automated installation script
- Manual installation from source
- Package managers (planned for future releases)
